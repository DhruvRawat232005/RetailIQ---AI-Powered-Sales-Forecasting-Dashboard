# RetailIQ – AI Powered Sales Forecasting Dashboard

RetailIQ is an interactive Business Intelligence dashboard that leverages Machine Learning and Time Series Forecasting to analyze retail sales, forecast future demand, detect sales anomalies, and perform product segmentation.

Built using **Python**, **Streamlit**, **Scikit-learn**, **SARIMA**, **XGBoost**, **Plotly**, and **Pandas**, RetailIQ transforms raw retail transaction data into actionable business insights through an intuitive and interactive dashboard.

# Features

### Sales Analytics
- Interactive KPI Dashboard
- Monthly Sales Trend Analysis
- Category-wise Sales Analysis
- Region-wise Sales Distribution
- Top 10 Selling Products
- Dynamic Filters (Region, Category & Customer Segment)

---

### AI Sales Forecasting

Implemented and compared multiple forecasting models:

- SARIMA
- Facebook Prophet
- XGBoost Regression

Model evaluation using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

The best-performing model is automatically selected to generate future sales forecasts.

---

### Sales Anomaly Detection

Implemented using **Isolation Forest** to identify unusual sales transactions.

Features include:

- Anomaly Count
- Anomaly Percentage
- Interactive Scatter Plot
- Outlier Identification

---

### Product Segmentation

Implemented using **K-Means Clustering**.

Segments products based on:

- Sales
- Order Count
- Shipping Duration

Provides:

- Cluster Summary
- Product Segmentation Visualization
- Cluster Distribution

---

### Interactive Dashboard

Built using **Streamlit** and **Plotly**

Features include:

- Interactive Charts
- Dark Theme UI
- Downloadable Reports
- Dynamic Sidebar Filters
- Responsive Layout
